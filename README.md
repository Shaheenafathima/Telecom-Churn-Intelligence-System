# AI-Powered Telecom Customer Churn Intelligence System

## Overview
This project predicts telecom customer churn using XGBoost and provides AI-powered retention strategies.

## Features
- Customer Churn Prediction
- SHAP Explainable AI
- Gemini AI Retention Strategies
- AI Chatbot
- Analytics Dashboard
- SQLite Database

## Technologies Used
- Python
- Flask
- XGBoost
- SHAP
- Gemini AI
- SQLite
- HTML, CSS, Bootstrap

## Installation

```bash
pip install -r requirements.txt
python app.py
```
